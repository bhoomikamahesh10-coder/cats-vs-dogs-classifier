# 🐱 Cats vs Dogs Classifier 🐶

This project uses a Convolutional Neural Network (CNN) built with TensorFlow and Keras to classify images as either *cats or dogs*.

## 🧠 Model Details

- Uses Conv2D, MaxPooling2D, Dropout layers
- Accuracy > 63% on validation set
- Includes data augmentation to prevent overfitting
